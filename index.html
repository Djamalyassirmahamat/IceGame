❄️ À propos d'IceGame

Bienvenue sur IceGame !

Une plateforme de jeux 100% gratuit avec 50 jeux amusants pour tous les âges.

🎮 Caractéristiques

  • ✅ 50 jeux différents
  • ✅ Système de score avec sauvegarde
  • ✅ Classement global et par jeu
  • ✅ Interface responsive (mobile, tablette, PC)
  • ✅ Entièrement gratuit
  • ✅ Aucune publicité

🎯 Les Jeux

Retrouvez :

  • 🐍 Jeux de stratégie (Snake, 2048)
  • 🎯 Jeux d'action (Flappy, Breakout)
  • 🧠 Jeux de mémoire (Memory, Matching)
  • 🎲 Jeux de hasard (Dés, Cartes)
  • ✏️ Jeux de logique (Tic-Tac-Toe, Quiz)

💾 Stockage Local

Tous vos scores sont sauvegardés sur votre appareil. Aucune donnée n'est envoyée à nos serveurs.

📱 Responsive Design

Jouez sur votre ordinateur, votre tablette ou votre téléphone mobile !
